# Agaate Website

A static HTML/CSS/JavaScript website for Agaate - Natural Farming & Technology-Driven Agriculture.

## Quick Start

### Run Locally

```bash
# Using Python 3
python3 -m http.server 8000

# Then open: http://localhost:8000
```

### Project Structure

```
Agaate/
├── components/     # Header and Footer components
├── data/          # Data files (brands, stats)
├── public/images/ # All images
├── index.html     # Main entry point
├── page.html      # Main content
├── page.js        # All functionality
└── layout.js      # Component loader
```

## Features

- ✅ Fully responsive design
- ✅ Header shrink on scroll
- ✅ Mobile menu
- ✅ Interactive tabs (Brands, Ecosystem, Testimonials)
- ✅ Image carousels
- ✅ Team marquee with 3D effects
- ✅ Smooth scroll navigation

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers

---

**Note**: This is a static website. You need a local server to run it (see Quick Start above).
